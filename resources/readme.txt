欢迎使用 LivoxViewerQT

LivoxViewerQT 是一款专为Livox激光雷达设备设计的Qt上位机软件，提供实时点云数据显示、设备配置和数据记录功能。

系统要求：
- Windows 10 或更高版本 (64位)
- Microsoft Visual C++ 2019 Redistributable (x64)
- 至少 4GB RAM
- 支持OpenGL 3.3或更高版本的显卡
- 网络连接（用于与激光雷达设备通信）

支持的设备：
- Livox MID-360
- Livox HAP (部分功能未适配）

主要功能：
- 实时点云数据显示
- 3D点云可视化
- 设备参数配置
- 数据记录和回放
- 网络配置管理
- 设备状态监控

安装说明：
1. 确保系统满足最低要求
2. 关闭防病毒软件（安装期间）
3. 以管理员身份运行安装程序
4. 按照安装向导完成安装

注意事项：
- 首次运行可能需要配置防火墙规则
- 确保激光雷达设备与计算机在同一网络段
- 建议使用有线网络连接以获得最佳性能
